# Finance Calculator MVP

Aplicación web para cálculos financieros básicos y avanzados.

## Características (Sprint 1)
- Calculadora Básica
- Interés Simple
- Interés Compuesto
- Historial y Exportación a CSV

## Uso
1. Abre `index.html` en tu navegador.
2. Selecciona la herramienta deseada.
3. Ingresa los datos y calcula.

## Desarrollo
- `src/js/`: Lógica de la aplicación.
- `src/css/`: Estilos.
- `tests/`: Tests unitarios.
